# 🏆 校園知識王 — 大屏觸控搶答遊戲

專為**大屏觸控電視（Windows）**設計的寓教於樂搶答遊戲，免安裝、免網路伺服器，雙擊即玩。

---

## 🚀 快速開始

1. 進入本資料夾，雙擊 `index.html`（建議使用 Edge 或 Chrome 開啟）。
2. 點右上角 `⛶` 按鈕進入全螢幕（大屏展示必按！）。
3. 選擇模式開始遊戲。

> 💡 大屏展示小技巧：可建立捷徑以 Kiosk 模式開啟，學生就無法跳出遊戲：
> `msedge.exe --kiosk "C:\...\index.html" --edge-kiosk-type=fullscreen`

---

## 🎮 遊戲模式

### 🚀 單人挑戰
10 題限時搶分（每題 15 秒）。計分公式：**100 基本分＋剩餘秒數×10＋連擊獎勵**，連續答對連擊越高、分數越高，營造「再挑戰一次」的動力。結束依答對率獲得 1–3 顆星，可選頭像登錄名人堂。

### ⚔️ 雙人對戰
螢幕左右分邊（粉紅隊 vs 藍藍隊），兩位學生同時站在大屏前搶答同一題。**先答對的隊伍得分；答錯會被鎖定該回合**，所以不能亂猜！共 8 回合，最適合班級競賽、下課挑戰賽。

### 👑 名人堂
單人挑戰的歷史前十名排行榜（儲存在本機瀏覽器，重開機仍保留）。

## ✏️ 如何擴充題庫（重點！）

題目全部放在 `questions.js`，用記事本打開即可編輯，**不需要懂程式**。

### 新增一題
找到想加的科目，照格式複製一行：

```js
{ q: "題目文字", options: ["正確答案","錯誤1","錯誤2","錯誤3"], answer: 0 },
```

- `answer: 0` 代表正確答案是第 1 個選項（0=第一個、1=第二個…）。
- 遊戲會自動打亂選項順序，所以正確答案都放第一個也沒關係。

### 新增整個科目
在 `QUESTION_BANK` 裡照格式加一段，遊戲畫面會**自動長出新科目按鈕**：

```js
"閩南語": {
  icon: "🗣️",
  color: "#B39DDB",
  questions: [
    { q: "「呷飽未」是什麼意思？", options: ["吃飽了嗎","睡飽了嗎","你好嗎","再見"], answer: 0 },
  ]
},
```

### 調整遊戲難度
打開 `game.js`，最上方有參數區可直接修改：

```js
const SINGLE_QUESTIONS = 10;   // 單人挑戰題數
const SINGLE_TIME      = 15;   // 單人每題秒數
const DUEL_ROUNDS      = 8;    // 雙人對戰回合數
const DUEL_TIME        = 20;   // 雙人每回合秒數
```

## ❓ 常見問題

- **沒有聲音？** 觸控螢幕任一處後音效才會啟動（瀏覽器安全限制），也可檢查右上角 🔊 是否被關閉。
- **排行榜想清空？** 在遊戲頁按 F12 → Console 輸入 `localStorage.removeItem("quizking_board")` 後重新整理。
- **字型跑掉？** 圓體字型來自 Google Fonts，離線時會自動改用微軟正黑體，不影響遊戲。
- **想改隊伍名稱？** 在 `index.html` 搜尋「粉紅隊」「藍藍隊」直接修改即可。
