# 🤖 AI 提示詞寶典

學習如何與 AI 有效溝通，打造專屬你的 AI 助手。

## 📚 內容

- **基礎概念** — 了解 Prompt 是什麼、為何重要
- **八大技巧** — 結構化、角色扮演、上下文等進階技巧
- **實用範本** — 20+ 常用提示詞範本
- **工具指南** — ChatGPT、Claude、Cursor 等工具使用技巧

## 🚀 使用方式

### 本地預覽
```bash
cd prompt-guide
python3 -m http.server 8080
```
然後瀏覽器打開 http://localhost:8080

### 部署
已部署至 GitHub Pages：https://hanclawbot.github.io/prompt-guide/

## 📝 編輯資料

修改 `data.json` 即可新增或編輯內容。
